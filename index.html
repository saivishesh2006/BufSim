BufSim — SQL Query Engine

Buffer pool simulation with LRU · MRU · CLOCK replacement policies.

Supported statements:

CREATE TABLE t (col INT, col CHAR(N))  ·  DROP TABLE t  ·  ALTER TABLE t ADD col TYPE

INSERT INTO t VALUES (...)  ·  UPDATE t SET col=val [WHERE ...]  ·  DELETE FROM t [WHERE ...]

SELECT [DISTINCT] cols FROM t [WHERE ...] [ORDER BY ...] [LIMIT n]

SELECT ... FROM t1 JOIN t2 ON t1.col = t2.col  ·  SHOW TABLES

Each SELECT / JOIN drives page-level buffer pool accesses. Buffer frame state updates on the right after every query.
